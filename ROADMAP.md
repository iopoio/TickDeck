- 🟡 **생성 결과 요약 카드** — 사용자가 분석 과정을 납득할 수 있도록

## 💰 요금 구조 설계 (초안)

### 무료 제공 (API 비용 없음)
| 기능 | 이유 |
|------|------|
| 텍스트 인라인 편집 | 프론트만 처리 |
| 슬라이드 순서 변경 | 프론트만 처리 |
| 컬러/레이아웃 오버라이드 | 캐시 재활용 |
| 슬라이드 삭제 | 프론트만 처리 |

### 크레딧 소모
| 기능 | 크레딧 |
|------|--------|
| 최초 생성 (크롤링+Gemini+Imagen) | 3 |
| 전체 재생성 (Gemini+Imagen, 크롤링 스킵) | 2 |
| 슬라이드 1장 재생성 | 0.5 |
| 이미지만 재생성 | 0.5 |

### 크레딧 충전
```
가입 시 무료 3크레딧 (1회 체험)
5,000원 = 10크레딧
10,000원 = 25크레딧 (20% 할인)
```

> 실제 API 비용: 회당 약 $0.20~0.50 (약 300~700원)
> 3크레딧 = 1,500원 → 마진 약 2배

```
1단계 (무료, 지금 바로): 재생성 버튼 → 결과 요약 카드 → 컬러 오버라이드
2단계 (크레딧 시스템): 레이아웃 변경 → 슬라이드 추가/삭제/재배열
3단계 (플랫폼화): 히스토리 → 공유 링크 → 언어 선택
```
