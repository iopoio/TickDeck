# TickDeck 품질 검사 체크리스트 (2026-03-30 배포분)

> 이 체크리스트를 https://tickdeck.site 에서 검증해주세요.
> KO = 한국어 버전 (/app), EN = 영문 버전 (/en/app)

---

## 1. 기본 접근성 (5분)

- [ ] https://tickdeck.site → 랜딩 페이지 정상 로드
- [ ] https://tickdeck.site/app → KO 앱 페이지 200 OK
- [ ] https://tickdeck.site/en/app → EN 앱 페이지 200 OK
- [ ] https://tickdeck.site/en → EN 랜딩 페이지 정상 로드
- [ ] KO 페이지에서 "🌐 EN" 클릭 → EN 페이지 이동
- [ ] EN 페이지에서 "🇰🇷 한국어" 클릭 → KO 페이지 이동

## 2. i18n 통합 검증 — KO (5분)

KO 앱 페이지 (/app)에서 확인:

- [ ] 페이지 제목: "TickDeck — 당신의 웹사이트를 PPT로"
- [ ] 히어로 타이틀: "당신의 웹사이트를 PPT로"
- [ ] 로그인 섹션: "시작하기" / "로그인하면 무료 토큰 2개가 지급됩니다"
- [ ] 구글 버튼: "Google로 계속하기"
- [ ] 네비게이션: "이력" 버튼, "로그아웃" 버튼
- [ ] 토큰 표시: "🎫 X개" (개 접미사)
- [ ] 폼 라벨: "홈페이지 주소", "회사명 (선택)"
- [ ] 용도 버튼: "✦ AI 추천", "브랜드 소개", "영업 제안" 등
- [ ] 생성 버튼: "슬라이드 생성하기"
- [ ] 하단 가이드: "1 웹사이트 분석", "2 AI 슬라이드 제작", "3 PPT 다운로드"
- [ ] 푸터: "💬 피드백", "☕ 후원"
- [ ] **깨진 텍스트 없음** ({{ t.xxx }} 같은 Jinja2 변수가 그대로 보이면 안 됨)
- [ ] **영문이 섞여 나오지 않음** (EN 텍스트가 KO 페이지에 등장하면 안 됨)

## 3. i18n 통합 검증 — EN (5분)

EN 앱 페이지 (/en/app)에서 확인:

- [ ] 페이지 제목: "TickDeck — Turn Your Website into a PPT"
- [ ] 히어로: "Turn Your Website into a PPT"
- [ ] 로그인: "Get Started" / "Sign in to get 2 free tokens"
- [ ] 구글 버튼: "Continue with Google"
- [ ] 네비게이션: "History", "Logout"
- [ ] 토큰 표시: "🎫 X" (개 없음)
- [ ] 폼: "Website URL", "Company Name (optional)"
- [ ] 용도: "✦ AI Pick", "Brand Intro", "Sales Proposal" 등
- [ ] 생성 버튼: "Generate Slides"
- [ ] 가이드: "Website Analysis", "AI Slide Creation", "PPT Download"
- [ ] 푸터: "💬 Feedback", "☕ Support"
- [ ] **한글이 섞여 나오지 않음**

## 4. 로그인 / OAuth 검증 (3분)

- [ ] Google 로그인 버튼 → Google 동의 화면으로 리다이렉트
- [ ] 로그인 성공 후 /app 페이지로 복귀
- [ ] 토큰 수 표시 정상
- [ ] 로그아웃 후 다시 로그인 페이지 표시

## 5. 모달 검증 (5분)

로그인 후:

- [ ] "이력"(History) 버튼 → 이력 모달 열림
- [ ] 이력 모달 닫기 정상
- [ ] 피드백 버튼(💬) → 피드백 모달 열림
- [ ] 피드백 카테고리 버튼 (🐛/💡/💬) 선택 가능
- [ ] 후원 버튼(☕) → 커피 모달/QR 표시

## 6. 슬라이드 생성 테스트 (10분)

토큰 있는 계정으로:

- [ ] URL 입력 (예: martinee.co.kr)
- [ ] "슬라이드 생성하기" 클릭
- [ ] 진행 표시: 4단계 dots 순서대로 진행
- [ ] 진행 텍스트 (KO: "🔍 웹사이트를 분석하고 있어요", EN: "🔍 Analyzing website")
- [ ] 완료 후 "슬라이드 생성 완료!" 표시
- [ ] PPTX 다운로드 정상
- [ ] PDF 다운로드 정상
- [ ] 슬라이드 미리보기 렌더링 정상
- [ ] 슬라이드 편집 버튼 → 편집 모달 열림
- [ ] 편집 저장 → 미리보기 업데이트
- [ ] AI 재생성 버튼 → 해당 슬라이드만 재작성

## 7. 슬라이드 품질 체크 (생성 결과물)

- [ ] 4, 5번 슬라이드가 **다른 레이아웃**으로 나옴 (연속 동일 방지)
- [ ] Contact(마지막) 슬라이드에 회사명 + 주소 표시
- [ ] CTA 슬라이드에 시간 약속 없음 ("X분 내" 같은 표현 없어야 함)
- [ ] 헤드라인에 "5가지"라 했으면 body가 정확히 5개

## 8. 보안 검증 (수동/기술 테스트)

- [ ] `/generate`에 `http://localhost:6379` URL → "내부 네트워크 URL은 사용할 수 없습니다" 에러
- [ ] `/generate`에 `http://192.168.1.1` URL → 동일 에러
- [ ] 로그인 없이 `/generate` POST → 401 에러
- [ ] `/api/auth/signup` 5회 초과 → 429 Rate Limit 에러
- [ ] `/admin` 비관리자 접근 → 403 에러

## 9. 에러 처리 (5분)

- [ ] 존재하지 않는 URL 입력 → 에러 메시지 (시스템 경로 노출 없어야 함)
- [ ] 에러 발생 시 "재시도" 버튼 표시
- [ ] 에러 상세 로그 토글 동작
- [ ] 토큰 0인 상태 → 생성 버튼 비활성 + "토큰이 부족합니다" 표시

## 10. 반응형 (모바일)

- [ ] 모바일 화면(375px 기준)에서 레이아웃 깨지지 않음
- [ ] 모바일에서 모달 정상 열림/닫힘
- [ ] 모바일에서 슬라이드 미리보기 스크롤 가능
